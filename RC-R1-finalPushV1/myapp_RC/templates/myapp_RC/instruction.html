INSTRUCTIONS

1. The reverse coding round will last for 30 minutes.
2. The answers are strictly in the form of integers.
3. Throughout this test, you'll be provided with three lifelines. You can use only a single lifeline for one question. These lifelines are described below.
4. Users are advised to code their solution in personal or external IDEs to compute the answer.

1

Astro Sub

This lifeline allows contestants to substitute a seemingly difficult question with an easier one that is still related to the same category. The easier question will have a marking scheme of +4 marks for a correct response to this easy question and -4 marks for a wrong response.
Marking Scheme:
+4 marks for correct response;
-4 marks for wrong answer.

2

Time Warp

This lifeline gives you the advantage of having additional time if you answer questions correctly. If you choose to use it, time will be added to the total test time, giving you more time to answer future questions. This lifeline can only be used once per contestant. For the first response, you get an additional benefit of 5 minutes if you answer the question correctly, but you lose 2 minutes in the case of a wrong response. If you answer it correctly in the second attempt, you get an additional 3 minutes, but you lose another 3 minutes if you give the wrong response.

Timer Scheme:
For first response ->
+5 min for correct response
-2 min for wrong response
For second response ->
+3 min for correct response
-3 min for wrong response

3

Galactic Guidance

This lifeline allows contestants to ask ChatGPT, the virtual assistant, for guidance and advice when unsure about the answer to a question. This lifeline can only be used after the contestant has answered three consecutive questions correctly. If they choose to use it, the marking scheme for that particular question will be adjusted to +4 for a correct answer and -4 for an incorrect answer, encouraging the contestant to use this lifeline wisely and at the right moment.
